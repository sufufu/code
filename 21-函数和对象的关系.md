### 什么是类

+ 内置类：Array Object RegExp Date Function（函数类）
+ Function 类：所有的函数都是 Function 类的实例；

- 内置类的数据类型：所有的内置类都是函数数据类型
console.log(typeof Array); // 'function'

所以，所有的内置类都是 Function 的实例。所以这些内置类有一个 __proto__ 属性，它的值指向 Function.prototype

// Object 和 Function 的关系

// 2. Function 是一个类，但是 Function.prototype 是一个对象，所以 Function.prototype 也有一个 __proto__ 的属性，它的值指向 Object.prototype。

// __proto__ 是实例用来找原型的，Object 是构造函数，Object.__proto__ 找原型，Object 是谁的实例呢，是 Function 的实例，Object.__proto__ 找的就是 Function 的 prototype

// Function.prototype 是对象，
